How entropy is calculated

Password: E = L × log₂(R)

L = length, R = pool size (unique characters)

R = uppercase(26) + lowercase(26) + digits(10) + symbols(32) + emojis(~3000)

Passphrase: E = W × log₂(7776)

W = word count, 7776 = EFF Diceware wordlist size

< 50 bits → Weak (brute-forceable)

50–79 bits → Fair (limited protection)

80–127 bits → Strong (secure for most uses)

≥ 128 bits → Military (quantum-resistant tier)
